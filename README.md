\# Primeira Atividade com Git



\## 📌 Sobre o Projeto

Este repositório foi criado como parte da minha primeira atividade prática utilizando \*\*Git\*\* e \*\*GitHub\*\*.  

O objetivo é aprender os conceitos básicos de versionamento de código e controle de alterações em projetos.



\## 🎯 Objetivos da Atividade

\- Compreender o funcionamento do Git

\- Configurar o ambiente de desenvolvimento

\- Criar e gerenciar um repositório

\- Realizar commits e registrar alterações

\- Enviar arquivos para o GitHub
